# 08 — A2A Simulado

## Objetivo
Simular comunicación con agente de otra organización.

## Agente externo
AgenteExternoSimulado expone un endpoint REST simple.

## Request ejemplo
{
  "consulta": "benchmark de conversión",
  "periodo": "2025-01"
}

## Response ejemplo
{
  "benchmark": {
    "conversion_promedio_industria": 0.031
  },
  "fuente": "organizacion_externa_simulada"
}

## Uso en flujo
El orquestador llama este agente en consultas comparativas.
