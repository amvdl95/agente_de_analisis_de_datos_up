# 05 — Agente de Respuesta

## Rol
Generar una respuesta clara para el usuario y persistir un resumen por MCP.

## Tool usada
mcp_escribir(destino, contenido)

## Estructura de respuesta
1. Resumen corto.
2. Hallazgos numéricos.
3. Predicción (si existe).
4. Comparación externa (si existe).
5. Limitaciones.

## Salidas en state
- reporte_final

## Requisito de cursada cubierto
- Escritura en sistema externo vía MCP.
